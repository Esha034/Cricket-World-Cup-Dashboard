# Cricket World Cup Dashboard

##  Project Overview
This project is a Business Intelligence dashboard built using **Google Looker Studio**.
It analyzes historical Cricket World Cup data to identify country-wise performance,
player comparisons, scoring trends, and venue-based insights.

The dashboard is designed as a **single-page BI report**, similar to industry dashboards
built in tools like Power BI and Tableau.

##  Dataset Details
The project uses the following datasets:

- Player performance dataset  
  (Runs, Balls, Strike Rate, Team, Opposition, Venue, Year)
- World Cup finals dataset  
  (Year, Winner, Winner Score, Final Venue)

All datasets are included in the `/data` folder.

---

## Dashboard Features
- KPI cards for high-level tournament insights  
- Country-wise World Cup wins comparison  
- Player vs Player comparison using side-by-side charts  
- Player performance comparison table  
- Runs trend analysis over years  
- Batting efficiency analysis using Runs vs Balls scatter plot  
- Geographical visualization of World Cup final venues
